### 任务一：表单（一）单个表单项的检验

demo： [点我](http://www.kumo-ka.com/code/IFE-2017/YYXY/task_1/index.html)

#### 任务目标

- 加强对JavaScript的掌握
- 熟悉常用表单处理逻辑

####  任务描述

- 如[示例图](http://7xrp04.com1.z0.glb.clouddn.com/task_2_29_1.jpg)中所示，在页面中实现一个输入框与按钮，要求点击验证按钮后，对输入框中内容进行格式校验，并在其下方显示校验结果
- 校验规则：
  - 字符数为4~16位
  - 每个英文字母、数字、英文符号长度为1
  - 每个汉字，中文符号长度为2
